Templates & Agent Market ​

Templates and the Agent Market are the best starting points for new users.

If you do not want to build an agent from scratch, start here.

Templates vs. Agent Market ​

These are two separate pages that serve different purposes.

Template Library ​

The Template Library shows templates that are already available locally.

You will see two categories:

  • Built-in — templates that ship with the app (General, Customer Service, Coding, Writing, Translation, Data Analysis)
  • Downloaded — templates you have downloaded from the Agent Market

Agent Market ​

The Agent Market is a discovery page for browsing and downloading templates created by the community.

Downloaded templates do not become agents automatically. They first go into your Template Library, and you then decide whether to create an agent from them.

Using the Template Library ​

From the Template Library page, you can:

  • Search templates by name
  • View a template's README
  • Create an agent from a template

Creating an Agent from a Template ​

  1. Select a template.
  2. Click Create Agent.
  3. Enter a name for the agent.
  4. Confirm.

The agent is created and appears in the sidebar. A platform model source is usually pre-selected automatically.

If you want to skip model selection during creation, you can assign one later in Agent Configuration.

README ​

Many templates include a README that explains:

  • What the template is designed for
  • Its default behavior
  • Whether additional skills or channels are required

It is worth reading the README before creating an agent.

Deleting Downloaded Templates ​

You can delete downloaded templates from the library. Deleting a template does not delete any agents that were already created from it.

Using the Agent Market ​

From the Agent Market page, you can:

  • Search for templates
  • View version, author, download count, and tags
  • Download templates to your local library

Template cards display localized names and descriptions based on your current app language.

After Downloading ​

  1. Click Download on a template card.
  2. The status changes to "Downloaded".
  3. The template appears in the Template Library under the Downloaded section.
  4. Go to the Template Library to create an agent from it.

Recommendation for New Users ​

If you just want to experience the full workflow:

  1. Open the Template Library.
  2. Pick a built-in template.
  3. Create your first agent.
  4. Once that is working, explore the Agent Market for community templates.

This approach is more reliable and avoids getting overwhelmed by choices early on.
