Quick Start ​

This page walks you through the first five minutes after installing ClawdSaber.

The Quick Path ​

  1. Log in.
  2. Complete the first-time setup wizard.
  3. Create your first agent.
  4. Pick a model for the agent.
  5. Start chatting.

Each step is explained below.

Step 1: Log In ​

Open ClawdSaber. You will see the login screen.

  1. Enter your email address.
  2. Click Send Magic Link.
  3. Check your inbox for an email from ClawdSaber.
  4. Click the link in the email, or copy and paste it into the app.

If the link opens your browser instead of the app, copy the full URL and paste it into the input field on the login screen.

For more details, see Login & Registration.

Step 2: First-Time Setup Wizard ​

After your first login, the setup wizard appears. It has three steps:

Welcome ​

Choose your preferred language (English, Chinese, or Japanese) and review a brief introduction to the app's key features.

Environment Check ​

The app verifies that the required runtime components are available:

  • OpenClaw — the backend engine that powers agents
  • Gateway — the local service that routes messages between you and your agents

If everything passes, you will see green checkmarks. If something fails, follow the on-screen instructions to resolve it.

Complete ​

A confirmation screen with links to the template library, agent marketplace, and settings. Click any of them to continue, or go straight to the main page.

The setup wizard only runs once. After completion, the app opens directly to the chat page on future launches.

Step 3: Create Your First Agent ​

You can create an agent from three places:

  • Template Library — Use a built-in or downloaded template. Best for getting started quickly.
  • Agent Market — Browse community-contributed templates, download one, then create an agent from it.
  • Sidebar "+" Button — A shortcut that opens the template picker.

Using a Built-In Template ​

  1. Open the Templates page from the sidebar.
  2. Pick a template from the Built-in section.
  3. Click Create Agent.
  4. Enter a name for your agent.
  5. Confirm.

The agent appears in the sidebar immediately.

Step 4: Pick a Model ​

A newly created agent does not have a model assigned yet. Without a model, it cannot respond to messages.

  1. In the sidebar, hover over your new agent and click the Settings icon.
  2. Go to the Model tab.
  3. Click Pick Platform Model.
  4. Select a model from the list. The dialog shows pricing information and a brief description for each model.
  5. Confirm your selection.

The model is now assigned. You can change it at any time.

Step 5: Start Chatting ​

  1. Click the agent in the sidebar to switch to it.
  2. Type a message in the input area at the bottom.
  3. Press Enter to send.

The agent streams its response in real time. You can stop generation at any time by clicking the Stop button.

Summary ​

The shortest path from install to first conversation:

  1. Log in with email magic link.
  2. Finish the 3-step setup wizard.
  3. Create an agent from a built-in template.
  4. Assign a platform model.
  5. Send your first message.

If the agent does not respond, check the FAQ.
