# Settings

The Settings page is where you manage your account, customize appearance, and control updates.

## Account

If you are logged in, the Account section shows:

- **Email** — the email address associated with your account
- **Nickname** — if you have set one
- **Plan** — your current subscription tier (e.g., Free)
- **Credit Balance** — how many credits you have remaining, with a **Recharge** button to top up
- **Plan Expiry** — when your current plan expires (if applicable)

### Actions

- **Refresh** — re-fetches your account info and credit balance from the server
- **Logout** — signs you out and returns to the login page

## Appearance

### Theme

Three options:

- **Light** — light background
- **Dark** — dark background
- **System** — follows your operating system's theme setting

### Language

Three languages are supported:

- English
- Chinese
- Japanese

The language change takes effect immediately across the entire app.

## About

A compact card between Appearance and Updates that shows:

- **App version** — the currently installed version of ClawdSaber
- **Website** — opens the official website
- **Telegram** — opens the Telegram community
- **Twitter** — opens the Twitter/X page

All links open in your default browser.

## Updates

### Current Version

Displays the version you are currently running, along with update status information.

### Update Controls

- **Check for Updates** — manually check whether a newer version is available
- **Download Update** — download an available update
- **Install & Restart** — install a downloaded update and restart the app

### Automatic Options

- **Auto-check for updates** — checks for updates every time the app starts
- **Auto-download updates** — downloads updates in the background automatically

Updates are downloaded in the background and installed when you restart the app.

## Top-Level Controls

Some controls appear in the title bar rather than the Settings page:

### Gateway Status Indicator

The title bar shows the current Gateway status:

- **Running** — the Gateway is active and ready
- **Loading** — the Gateway is starting up
- **Error** — something went wrong; check the FAQ
- **Stopped** — the Gateway is not running

### Memory Mode

A brain icon in the title bar toggles memory mode. This is an optional feature for advanced users that enables the agent to recall information across sessions.
